# 🛒 E-Commerce Analytics Dashboard

## 📌 Overview

This project is an interactive **E-Commerce Analytics Dashboard** built using **Streamlit, MySQL, and Python**.
It provides insights into sales performance, customer behavior, and business trends through visualizations and KPIs.

---

## 🚀 Features

### 📊 KPI Metrics

* Total Customers
* Total Revenue
* Total Orders

### 🔍 Data Exploration

* Filter customers by city
* Search customers by name
* Download customer data as CSV

### 📈 Visualizations

* Sales by Product (Bar Chart)
* Monthly Revenue Trend (Line Chart)
* Order Status Distribution

### 🏆 Business Insights

* Top Revenue Generating Product
* Top Customers by Spending
* Least Selling Product
* Top Customer City
* Most Used Payment Method
* Revenue Trend Analysis (Increasing/Decreasing)

---

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** MySQL
* **Libraries:** Pandas

## ⚙️ Setup Instructions

### 1. Clone the Repository

```
git clone https://github.com/rajanandini1224/ecommerce-dashboard.git
cd ecommerce-dashboard
```

### 2. Create Virtual Environment (Optional)

```
python -m venv .venv
.venv\Scripts\activate
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Run the Application

```
streamlit run app.py
```
